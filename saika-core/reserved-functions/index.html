Reserved Functions

Just like other frameworks, Saika has some reserved functions. Saika's core functions are located in system/core/Functions.php. The file loads when Saika boots up. Also Saika uses the functions in it's core classes. So, you can't replace the core functions.

Saika's Reserved Functions

saika_detect_url()

  • Detects site URL
  • @return string

site_url($path)

  • Quick access the Site URL
  • @param string $path (Optional) The path
  • @return string

load_helper($helper_file_name)

  • Load user's custom defined functions aka helpers
  • @param string $helper_file_name The helper file name ( without .php )
  • @return

get_random_string($length = 10, $keyspace = '0123456789abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTWXYZ')

  • Get a random string. Not secure! Just a random string!
  • @param integer $length The length of the string
  • @param string $keyspace The characters to use, Default is: 0-9-A-z
  • @return string

get_secure_token($length = 20)

  • Generate a cryptographically secure random token
  • @param integer $length The length of token ( may differ because it will be converted to hex )
  • @return string

get_current_url()

  • Get current URL
  • @return string

saika_url_slug($str, array $options = array())

  • Generate SEO friendly URL slug from string. With Unicode support.
  • @param string $str The string
  • @param array $options The options as array, the option keys are:
    • delimiter : URL delimiter, default is "-"
    • limit : how long the slug could be? default is "null"
    • lowercase : whether to convert it to lowercase or not
    • replacements : custom replacements, as key => val array
    • transliterate : Transliterate characters to ASCII or not
